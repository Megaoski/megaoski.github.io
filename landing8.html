GameJam: Mae

Game done for the GRANCITM GAMEJAM 2021

Summary

Mae is a game developed in 36h for the GRAN CITM GAMEJAM 2021. The theme for the gamejam was to recognize the women working(or worked) in S.T.E.A.M. We chose to do a game tribute to Mae C. Jemison the first afroamerican woman in space.

My role in the project

Since Mae is a gamejam project that I worked with 3 more members on for 36h, I would say I worked on all of the game aspects since we had little time to finish the game.
