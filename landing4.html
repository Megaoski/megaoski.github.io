AI Dionysius

AI game using Unity

Summary

This project is an AI exercise developed in Unity to achieve the Wander & Patrol behaviours using Behaviour Bricks. It includes the posibility of adding until 16 tanks and 4 different teams to recreate a fight between them and observe which one is the winner.

My role in the project

Developed the Wander Behaviour script
Developed the Patrol Behaviour script
Implementing Behaviour Bricks to the project
